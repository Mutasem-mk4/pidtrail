# Security Policy

## Scope

`pidtrail` is for authorized inspection of processes and commands you control or are explicitly permitted to investigate.

## Security posture

- No telemetry
- No analytics
- No background daemon
- Metadata-focused capture surface
- No plaintext TLS claims
- No install-time capability changes

## Reporting

Please report vulnerabilities privately before public disclosure. Include:

- affected commit or release
- distribution and kernel version
- command line used
- whether the failure was in process, file, or network tracing
- minimal reproduction steps
